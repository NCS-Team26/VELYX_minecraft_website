친구 초대용 접속 안내

이 페이지 링크만 보내면 모드팩 설치부터 서버 접속까지 한 번에 안내됩니다.

velyx.kr

서버 주소는 Minecraft Java 멀티플레이에 그대로 넣으면 됩니다.

1 Modrinth App 설치

처음 접속하는 친구는 Modrinth App 또는 Prism Launcher가 가장 빠릅니다.

2 모드팩 가져오기

.mrpack 파일을 가져오면 Fabric Loader와 필수 모드가 자동으로 맞춰집니다.

3 프로필 실행

VELYX Fabric 26.1.2 또는 가져온 Modrinth 프로필로 실행합니다.

4 서버 접속

Minecraft Java 멀티플레이에서 velyx.kr을 서버 주소로 저장합니다.

3분 설치

친구는 여기 버튼만 누르면 됩니다

추천은 Modrinth App으로 모드팩을 가져오는 방식입니다. 기본 Minecraft Launcher만 쓰는 친구는 Windows 자동 설치 스크립트를 받아 실행하면 됩니다.

PowerShell 스크립트가 Fabric 프로필과 모드 16개를 .minecraft에 설치합니다.
